# Solution Log Writer Skill

## Purpose

본선 PT와 재현성 검증을 위해 문제 해결 과정을 기록한다.

## Inputs

- generated/contest_spec.yaml
- generated/gap_report.md
- runs/*/run_log.json
- outputs/submission.csv

## Outputs

- generated/solution_log.md
- 발표자료 초안에 들어갈 구조 설명

## Procedure

1. 문제 이해를 요약한다.
2. 데이터 구조와 제출 형식을 기록한다.
3. 하네스 구조를 설명한다.
4. 실험별 solver/config/결과를 표로 기록한다.
5. 실패한 실험도 기록한다.
6. 최종 선택 이유와 규칙 준수 근거를 기록한다.
